# Overview

This is a collection of examples with STM32F102C6T8 (in some examples used STM32F103C8T6). 

Examples are arranged in order of increasing difficulty.

The examples are arranged in order of increasing complexity and are independent of each other. To build each example, go to the example folder and check out the file README.md

| Folder | Example description |
|:----------|:--------|
| 1_STM32F103C6T8_minimal_C | Minimal application to run code in C language |
| 2_STM32F103C6T8_minimal_GPIO | GPIO (MCU pins) demo. How to use the general purpose I/O pins to output a digital signal to control an LED and read pin values ​​generated by a pushbutton. Based on CMSIS |
